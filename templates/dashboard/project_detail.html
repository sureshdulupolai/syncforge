Integration Guide

Zero-code synchronization for Django applications.

Install the SDK Available on PyPI. Zero external dependencies.
1
Install the SDK
Terminal
# Install via pip
pip install syncforge
Create sf_client.py Initialize the client with your key.
2
Create sf_client.py
sf_client.py
import os
from syncforge import SyncForge

# Initialize client for {{ project.name }}
sf = SyncForge(
    api_key=os.environ.get('SYNCFORGE_API_KEY', '{{ api_keys.first.key_prefix|default:"sf_live_" }}...')
)
Zero-Code Syncs Add the decorator to your models.
3
Zero-Code Syncs
models.py
from django.db import models
from sf_client import sf
from syncforge.django import sync_model

# Auto-registers table & syncs on all DB changes
@sync_model(sf)
class Item(models.Model):
    name = models.CharField(max_length=100)
    price = models.IntegerField()
