Integration Guide

1

Install SyncForge

# pip
pip install syncforge

# npm
npm install syncforge
2

Initialize with your API Key

from syncforge import SyncForge

# Dead simple — like FastAPI
app = SyncForge(api_key='sf_live_your_key')
3

Push updates instantly

def save_product(data):
    Product.objects.create(**data)
    # Push delta to all clients ⚡
    app.push('products')
    return 'Synced instantly — zero polling'
✓

Or use the REST API directly

# Trigger sync via HTTP — works from any language
curl -X POST {{ request.scheme }}://{{ request.get_host }}/api/v1/sync/products/ \
    -H "X-API-Key: sf_live_your_key"

# Health check
curl {{ request.scheme }}://{{ request.get_host }}/api/v1/health/
