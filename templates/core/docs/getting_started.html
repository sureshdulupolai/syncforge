Overview

Getting Started

Install the SDK, create a project, get your API key, and make your first sync call — all in under 5 minutes.

1 Install the SDK

terminal 
pip install syncforge

No external dependencies. Pure Python Standard Library.

2 Create a project & get your API key

  1. → Register at syncforge.dev
  2. → Create a new project in the dashboard
  3. → Copy your API key — it starts with sf_live_
  4. ⚠ The raw key is shown only once — store it in your .env immediately

3 Set your environment variable

.env 
SYNCFORGE_API_KEY=sf_live_your_key_here
Never hardcode your API key in source code. Always read it from environment variables. Your .env file must be listed in .gitignore.

4 Initialise the client

Create one SyncForge instance per project. Place it in a module-level file and import it wherever needed.

syncforge_client.py 
import os
from syncforge import SyncForge

sf = SyncForge(api_key=os.environ['SYNCFORGE_API_KEY'])

5 Signal a data change

Call sf.refresh() after every database write. SyncForge will invalidate cached data and update the server.

example.py 
from syncforge_client import sf

# After any DB write
product = Product.objects.create(name='Widget', price=9.99)
sf.refresh('products')   # Tell SyncForge this table changed

✓ Verify connectivity

python 
if sf.ping():
    print('SyncForge connected!')
else:
    print('Check your API key or network')
